# 🎬 MovieApp - Responsive Landing Page

This project is a single-page landing interface for a fictional movie/TV show platform, developed using modern web design standards.

## 🚀 About the Project

The main purpose of this study is to grasp semantic HTML structures, apply the **BEM (Block, Element, Modifier)** naming methodology, and build a modern, flexible, and responsive interface using **CSS Flexbox**. The page is designed to work seamlessly across all devices, from large desktop monitors to mobile screens.

## 🛠️ Technologies & Concepts Used

- **HTML5:** Structured using semantic tags (`<nav>`, `<section>`, `<main>`, `<footer>`) to comply with SEO and accessibility standards.
- **CSS3:**
  - Modern flexible box layout architecture (**Flexbox**)
  - Clean, organized, and sustainable class naming (**BEM Methodology**)
  - Full mobile responsiveness via media queries (**Media Queries / @media**)
  - Sticky navigation menu (`position: sticky`) and smooth page scrolling (`scroll-behavior: smooth`)
  - Modern card elevations (**Box Shadows**) and smooth interaction states (**Transitions/Hover effects**)

Bu çalışmanın temel amacı; semantik HTML yapısını kavramak, **BEM (Block, Element, Modifier)** isimlendirme metodolojisini uygulamak ve **CSS Flexbox** ile modern, esnek ve mobil uyumlu (responsive) bir arayüz inşa etmektir.
